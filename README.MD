# CHIP-8 Emulator
**Written in C++ and using the SDL2 library.**

## Build instructions
### Dependencies
- `g++`
- `cmake`
- `SDL2`


### Build command
#### Configure build
```
$ mkdir build
$ cd build
$ cmake ../
```
#### Build debug version
```
$ cmake --build /home/kunal/Development/CHIP8Emu/build --config Debug --target all
```
